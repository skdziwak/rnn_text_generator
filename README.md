# rnn_text_generator
Python scripts that are able to learn and fake a natural language based on Recurrent Neural Networks

## Usage examples:

### Train
```
python train.py -tp data.txt -ep 15 -ed 512 -ru 2048 -i rnn_info.json
```

### Generate text
```
python generate.py -i rnn_info.json -s Hello -l 1200 -t 1.3 -o generated.txt
```

## Help:

### train.py --help

```
usage: train.py [-h] [--epochs EPOCHS]
                [--embedding-dimension EMBEDDING_DIMENSION]
                [--rnn-units RNN_UNITS] [--buffer-size BUFFER_SIZE]
                [--batch-size BATCH_SIZE] [--sequence-length SEQUENCE_LENGTH]
                [--info INFO] [--early-stopping EARLY_STOPPING]
                [--training-path TRAINING_PATH]
                [--checkpoints-path CHECKPOINTS_PATH]

optional arguments:
  -h, --help            show this help message and exit
  --epochs EPOCHS, -ep EPOCHS
                        Epochs (default: 10000)
  --embedding-dimension EMBEDDING_DIMENSION, -ed EMBEDDING_DIMENSION
                        Embedding layer output dimension (default: 256)
  --rnn-units RNN_UNITS, -ru RNN_UNITS
                        GRU layer units (default: 1024)
  --buffer-size BUFFER_SIZE
                        Dataset shuffling buffer size (default: 1000)
  --batch-size BATCH_SIZE
                        Training batch size (default: 32)
  --sequence-length SEQUENCE_LENGTH, -sl SEQUENCE_LENGTH
                        Training batch size (default: 150)
  --info INFO, -i INFO  Path where info file will be saved (default:
                        info.json)
  --early-stopping EARLY_STOPPING, -es EARLY_STOPPING
                        Early stopping patience (default: 0, no early
                        stopping)
  --training-path TRAINING_PATH, -tp TRAINING_PATH
                        Path to training file or directory of training files
                        (default: training.txt)
  --checkpoints-path CHECKPOINTS_PATH, -cp CHECKPOINTS_PATH
                        Path to directory where weights will be saved
                        (default: checkpoints)

```


### generate.py --help
```
usage: generate.py [-h] [--info INFO] [--output OUTPUT] [--start START]
                   [--length LENGTH] [--temperature TEMPERATURE]
                   [--print-output PRINT]

optional arguments:
  -h, --help            show this help message and exit
  --info INFO, -i INFO  Path to info file generated by train.py (default:
                        info.json)
  --output OUTPUT, -o OUTPUT
                        Path to output file (default: None, prints output)
  --start START, -s START
                        Start of generated string (default: None, random
                        character)
  --length LENGTH, -l LENGTH
                        Length of generated text (default: 512)
  --temperature TEMPERATURE, -t TEMPERATURE
                        Temperature (default: 1.0)
  --print-output PRINT, -p PRINT
                        Print output (default: true)
```
